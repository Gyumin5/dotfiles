# /project-review: 프로젝트 CLAUDE.md 점검 및 보완

현재 프로젝트의 CLAUDE.md를 점검하고, 빠진 내용을 찾아 보완을 제안하라.

## 단계

### 1. 현재 CLAUDE.md 읽기

- 프로젝트 루트와 서브디렉토리의 CLAUDE.md를 모두 읽어라

### 2. 프로젝트 실태 분석

아래 항목들을 실제 프로젝트 파일과 대조하여 누락된 것을 찾아라:

- 빌드 시스템이 있는데 Build 섹션이 없는지
- 테스트 프레임워크가 있는데 Test 섹션이 없는지
- 린터/포매터 설정 파일이 있는데 Style 섹션이 없는지
- launch 파일, config 파일 등 건드리면 안 되는 파일이 보호되지 않는지
- 주요 디렉토리 설명이 빠져있는지

### 3. 보완 제안

- 누락된 항목을 구체적으로 제안하라
- 각 제안에 "추가할까요?" 확인을 받아라
- 승인된 항목만 CLAUDE.md에 추가하라

### 4. 불필요한 항목 정리

- 더 이상 해당되지 않는 규칙이 있는지 확인
- 중복된 규칙이 있으면 병합 제안
